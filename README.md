# Xignis App

App mobile-first para empleados y panel web responsive para RH/admin.

## Stack

- React + Vite + TypeScript
- Capacitor
- Tailwind CSS
- Supabase

## Comandos

```bash
npm install
npm run dev
npm run build
npm test
npm run cap:sync
```

## Configuracion

Crear `.env` con:

```bash
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```
